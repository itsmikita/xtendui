## Customization

You can **customize this component** inside `tailwind.config.js`, check [xtendui/src/addons/nav-sides.css.js](https://github.com/minimit/xtendui/blob/master/src/addons/nav-sides.css.js) for default values.

```jsx
module.exports = {
  theme:{
    xtendui: {
      'nav-sides': {
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}
```
