## Setup

To use this component import the **less** and **js** files accordingly:

```less
@import '~xtend-library/src/core/toggle/toggle.less';
```

```jsx
import 'xtend-library/src/core/toggle/toggle.js'
```

Or just import **core**:

```jsx
import 'xtend-library/src/xtend-core.js'
```
