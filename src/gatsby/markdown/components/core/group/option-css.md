## Variant

Classes for assigning variant (e.g.: border / background / color). Add or remove selectors at your discretion.

<div class="table-scroll">

|                         | Syntax                                    | Example                       |
| ----------------------- | ----------------------------------------- | ----------------------------- |
| Class                   | `.group-{variant}`                        | `.group-default`             |
| Mixin                   | `.group-{variant}()`                      | `.group-default()`            |

</div>
