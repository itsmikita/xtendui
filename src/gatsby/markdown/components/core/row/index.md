## Customization

You can **customize this component** inside `tailwind.config.js`, check [xtendui/src/core/row.css.js](https://github.com/minimit/xtendui/blob/master/src/core/row.css.js) for default values.

```jsx
module.exports = {
  theme:{
    xtendui: {
      row: {
        utility: theme => ({
          // modify utilities
        }),
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}
```
