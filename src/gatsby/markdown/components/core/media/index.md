## Customization

You can **customize this component** inside `tailwind.config.js`, check [xtendui/src/core/media.css.js](https://github.com/minimit/xtendui/blob/beta/src/core/media.css.js) for default values.

```jsx
module.exports = {
  theme:{
    xtendui: {
      media: {
        utility: theme => ({
          // modify utilities
        }),
        component: theme => ({
          // modify components
        }),
      },
    },
  },
}
```
