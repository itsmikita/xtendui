Initialization

All

You can initialize all javascript components on the document attributes (using [data-xt-]) using XtUtil. Be sure to initialize them when the page is ready.

  
    function initAll() {
      XtUtil.initAll();
    }
    if (document.readyState !== 'loading') {
      initAll();
    } else if (document.addEventListener) {
      document.addEventListener('DOMContentLoaded', initAll);
    }

You can also pass a container to search for javascript components.

  
    XtUtil.initAll(document.documentElement.querySelectorAll('.my-container'));

Toggle

You can initialize toggle by javascript.

  
    // single
    new XtToggle(document.documentElement.querySelectorAll('.my-element')[0], {
      // options
    });

    // looping
    Array.from(document.documentElement.querySelectorAll('.my-element')).forEach(function (el, i) {
      new XtToggle(el, {
        // options
      });
    });
