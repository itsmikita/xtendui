Polyfill

For supporting older browsers you need to add babel's polyfill.js, see docs, cdn.

Initialization

All

You can initialize all javascript components on the document attributes (using [data-xt-]) using Xt. Be sure to initialize them when the page is ready.

  
    function initAll() {
      Xt.initAll();
    }

    if (document.readyState === "loading") {
      document.addEventListener('DOMContentLoaded', initAll);
    } else {
      initAll();
    }

You can also pass a container to search for javascript components.

  
    Xt.initAll(document.querySelectorAll('.my-container'));

Toggle

You can initialize toggle by javascript.

  
    // single
    new XtToggle(document.querySelectorAll('.my-element')[0], {
      // options
    });

    // looping
    for(let [i, el] of document.querySelectorAll('.my-element').entries()) {
      new XtToggle(el, {
        // options
      });
    }
