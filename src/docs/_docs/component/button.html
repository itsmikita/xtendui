Links

On containers use .links-inverse to have white links.

In the code the mixins .link-underline() and .link-no-underline() are used assign proper underline on buttons.

Buttons

Every button must have the .btn class.

Variants

You need to specify a variant on each .btn. -empty variants are for buttons without background.

.btn-link is styled like normal links but with custom styles.

.btn-primary .btn-primary-empty .btn-secondary .btn-secondary-empty .btn-link

Sizes

Optional classes for assigning sizes. By default .btn-medium is used on all .btn. Sizes uses em padding that depends on font size.

.btn-tiny .btn-small .btn-medium .btn-big .btn-giant

Special

Special optional modifiers classes.

.btn-squared .btn-none .btn-icon .btn-tall .btn-short .btn-wide .btn-narrow
