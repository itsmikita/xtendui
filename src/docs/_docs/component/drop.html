Usage

Use this markup to create a dropdown. [data-xt-drop] must be initialized by javascript.

For customizing the interactions read the documentation of toggle drop.

Drop of anything

You can use drop to make a dropdown with inside anything. For example we use a box.

Lorem ipsum dolor dolor sit amet, consectetur adipiscing elit.
Nullam posuere ex ac felis venenatis pretium.
Aenean aliquam interdum enim et faucibus.
In tempor convallis est vel dapibus.

Drop of lists

The natural way to use drops is to use them with lists of buttons. Read the documentation of list of buttons.

Options

Variants

Optional classes for assigning variant to style all buttons the same way.

.list-drop has drop's list default options.

.drop-default
.drop-primary
.drop-secondary
.drop-white
You can override each buttons by using the .btn class. For custom list style don't use .list-drop.

Special

Special optional modifiers classes.

.drop-squared
.drop-noborder

Dimensions

Drop with max sizes with automatic scrollbar on overflow.

.drop-max-height
.drop-max-width
.drop-max-height .drop-max-width

Position

Optional classes for assigning position.

.drop-bottom .drop-left .drop-right .drop-center
.drop-top .drop-left .drop-right .drop-center
.drop-middle .drop-left .drop-right .drop-center
.drop-before .drop-top .drop-bottom .drop-middle
.drop-after .drop-top .drop-bottom .drop-middle
.drop-middle-top .drop-middle-bottom

Static

Drop position relative to closest element with position: relative;.

.drop-container-static .drop-bottom
.drop-container-static .drop-top
.drop-container-static .drop-middle

Read the documentation of component's variables.
