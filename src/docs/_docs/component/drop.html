Usage

Use this format to create a drop. For customizing the interactions you can read the documentation of toggle drop.

Drop of anything

You can use drop to make a dropdown with inside anything. For example we use a box.

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
Nullam posuere ex ac felis venenatis pretium.
Aenean aliquam interdum enim et faucibus.
In tempor convallis est vel dapibus.

Drop of lists

The natural way to use drops is to use them with lists of buttons. Read the documentation of list of buttons.

Options

List

You also have to assign a list variant.

.list-drop has drop's list default options that you can override adding classes, except the // not overridable options in the theme.

.list-drop
.list-space-tiny .list-center .list-primary-empty .list-big .list-short

Variants

Optional classes for assigning variant.

.list-drop
.drop-white .list-primary-empty
.drop-primary .list-primary
.drop-secondary .list-secondary

Special

Special optional modifiers classes.

.drop-squared
.drop-noborder
