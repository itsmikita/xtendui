Usage

Use this markup to create a fade.

You can initialize fade by javascript omitting [data-xt-fade].

  
    let self = Xt.init('xt-fade', document.documentElement, {
      // option
    });
    self.unmount = function() {
      self = null;
    };
It's recomended to animate the fade elements with animation instead of transition because of this bug.

Direction and Distance

.fade--up .fade--down
"distance": Number is the window's top and bottom activation and deactivation. Factor between 0 and 1

Timing

You can use a function for "delayOn" and "delayOff" like this: function(current, total) {return current * 75}. You can also use a string "return current * 75".
