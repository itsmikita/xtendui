Usage

Use this markup to create a fade. [data-xt-fade] must be initialized by javascript.
Instead of using html arguments you can initialize fade by javascript using the same options syntax.
It's recomended to animate the fade elements with animation instead of transition because of this bug.

Javascript

Example initialized by javascript on document

Defaults

Here are the defaults options for fade.

  
Xt.Fade.defaults = {
  "elements": ".fade",
  "class": "in",
  "on": "scroll resize",
  "min": 0,
  "max": Infinity,
  "distance": 0.2,
  "delayOn": false,
  "delayOff": false,
  "aria": false
};
