Options

Group

With "groupMq": {MediaQuery: Number} you can set a decimal number from 0 to 1 for the available width factor for groupMq. You can also set multiple with media queries.

If a slide is bigger, it takes up a single slide anyway.

it's based on the current slides visible inside the slider, it get recalculated also on responsive.

"groupMq": {MediaQuery: Number}
"groupMq": false to disable automatic grouping of slides

Align

"align": "center"
"align": "left"
"align": "right"

Contain

Use "contain": true to contain the slides.

"contain": true "align": "center"
"contain": true "align": "left"
"contain": true "align": "right"

Navigation and Pagination

You can add navigation with [data-xt-nav], set the amount to add to the current activation with [data-xt-nav="Number"]

You can add pagination inside .slider_pagination with .xt-clone to generate the pagination, inside it you can use this variables: {% raw %}{{num}}{% endraw %} {% raw %}{{tot}} {% endraw %}.

Pagination is essential to the functioning of the slider, so if you don't want to show it add .display--none to .slider_pagination.

Wheel, auto and disabled

Use "wheel": {"selector": "object"} or "wheel": {"selector": Query} to enable mousewheel navigation.

For auto options see toggle auto.

To disable slider use .slider--disabled useful within media queries.

Progress

Use events 'start.xt.auto' 'stop.xt.auto' 'pause.xt.auto'

Other

.active "loop": false "jump": false "autoHeight": false

Drag and toggle

Use "drag": false to disable drag.

You can use the slider in toggle mode, so with .toogle--block on the slides.
