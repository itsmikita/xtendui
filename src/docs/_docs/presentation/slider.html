Usage

Use this markup to create a drop. [data-xt-drop] must be initialized by javascript.
Instead of using html arguments you can initialize slider by javascript using the same options syntax.
We use javascript animations with tweenlite and customease

Options

Initial

Just assign the activation class on any target

Auto

see toggle auto "auto": Milliseconds "autoChange": Milliseconds "autoPause": String "autoInverse": Boolean

Alignment

"dragAlig": "center"
"dragAlig": "left"
"dragAlig": "right"

Loop

"loop": false to disable loop (also on automatic)
