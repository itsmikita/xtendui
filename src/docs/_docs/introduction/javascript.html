Polyfill

For supporting older browsers you need to add babel's polyfill.js, see docs, cdn.

Initialization

Attributes

The document attributes with [data-xt-] gets initialized automatically, also on DOM changes with Mutation Observer.

If you really need, but probably not, you can call yourself the document attributes initializzation.

  
    Xt.initAll(document.querySelectorAll('.my-container'));

Toggle

You can initialize toggle by javascript.

  
    // single
    Xt.Toggle(document.querySelectorAll('.my-element')[0], {
      // options
    });

    // looping
    for (let [i, el] of document.querySelectorAll('.my-element').entries()) {
      Xt.Toggle(el, {
        // options
      });
    }
