Usage

Use this markup to create a sticky. [data-xt-sticky] must be initialized by javascript.
Instead of using html arguments you can initialize sticky by javascript using the same options syntax.
It's recomended to style and animate a inner div [data-xt-xticky] > div when using sticky component.

Defaults

Here are the defaults options for sticky.

  
    XtSticky.defaults = {
      "class": "active",
      "on": "scroll",
      "min": 0,
      "max": Infinity
    };

Options

Position and Mode

The default position is "position": "top".

The default limit for "limit": {"top": Element} is the element's own position, for "limit": {"bottom": Element} is Infinity.

You can optionally set in the less mixin a mode fixed or absolute, to have that position when not sticky.

"position": "top" "position": "bottom".
"limit": {"top": Element} "limit": {"bottom": Element}.
Use "mode": "fixed" in the less to have fixed position when not sticky.
Use "mode": "absolute" in the less mixin to have absolute position when not sticky.

Contain

Use the "contain": {} option to stack consecutive sticky elements. with "contain": {"top": Element} you stack the sticky to a previous element, with "contain": {"bottom": Element} you stack the sticky to a next element. The two solutions brings different stacking effects.

"contain": {"top": Element}.
"contain": {"bottom": Element}.
"hide": "down" "hide": "up".
