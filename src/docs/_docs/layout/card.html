Sizes

Optional classes for assigning size. Sizes uses em padding that depends on font size.

You can change the default size inside the .card selector in /theme/_layout.less.

Card has margins specified in the size classes. Additionally it has .margin(y, layout) for no margins :first-child or :last-child
