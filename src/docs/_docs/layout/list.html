--- title: List categories: [Layout] order: 300 description: List is used to space it's childrens in a column/row layout. Can also be used to automatically style lists of buttons. ---

Usage

Use .list to create a row list or .list.list--block to create a column list with flexbox.

Every list should have the ul tags, with li childrens.

List of buttons

The natural way to use lists is to use them to space buttons.

You can style also dropdown's buttons by assigning the .drop_outer to the .list > li

Buttons styles

.list > li > a and .list > li > button are styled automatically, If you want to customize the buttons use the .btn class instead.

In theming code use .list_btn--default({}); to style a and button. To style also if they have the .btn class use .list_btn--all({});.
You can override each buttons by using the .btn class

Options

Variants

Optional classes for assigning variant.

You can change the default variant inside the .list and .list--block selectors in /theme/_layout.less.

.list--primary
.list--primary-empty
.list--secondary
.list--secondary-empty
.list--link

Space

Lists automatically add horizontal and vertical spacing to the contents. You can specify a different one with this classes/mixins.

Syntax Example
Class .list_space--{size} .list_space--small
Class responsive .list_space--{size}-{breakpoint} .list_space--small-sm
Mixin .list_space({size}) .list_space(small)
Mixin responsive .list_space({size}, {breakpoint}) .list_space(small, sm)
You can set list's spaces in the _variables.less file. Read the documentation of list's variables.
.list_space--none
.list_space--tiny
