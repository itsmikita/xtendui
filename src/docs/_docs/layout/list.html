Usage

Use .list-block to create a column list or .list-line to create a row list with flexbox.

Every list should have the ul tags, with li childrens.

List of buttons

The natural way to use lists is to use them to space buttons in a column/row. You can specify for each button variant, size and all the properties you want.

Options

Sizes

Lists automatically add horizontal and vertical spacing to the contents. You can specify a different one with this classes/mixins.

Syntax Example
Class .list-space-{size} .list-space-small
Class responsive .list-space-{size}-{breakpoint} .list-space-small-sm
Mixin .list-space({size}) .list-space(small)
Mixin responsive .list-space({size}, {breakpoint}) .list-space(small, sm)

You can set list's sizes _variables.less file. Read the documentation of list's variables.

.list-space-none
.list-space-tiny
.list-space-small
.list-space-medium
.list-space-big
.list-space-giant

Alignment

Lists automatically add left aligment. You can specify a different one with this classes/mixins.

Syntax Example
Class .list-left .list-left
Class responsive .list-left-{breakpoint} .list-left-sm
Mixin .list-left() .list-left()
Mixin responsive .list-left({breakpoint}) .list-left(sm)
.list-left
.list-center
.list-right

List options

List options are used to style to all the buttons inside (a or button) with a default option, without the need to assing for each one the classes.

You have a list option for each option of .btn. .list-default is a default option.

Variants

Optional classes for assigning variant.

.list-default
.list-primary
.list-primary-empty
.list-secondary
.list-secondary-empty
.list-link

Sizes

Optional classes for assigning size.

.list-tiny
.list-small
.list-medium
.list-big
.list-giant

.list-narrow

.list-none
