--- title: Flex categories: [Layout] order: 200 description: Flex is used to place it's childrens in a column/row layout. You can set and all the flex's properties using utils classes or mixins. ---

Read the documentation of Layout with theming and usage info.

Usage

Use .flex to create a column/row layout with flexbox.

You have to use columns inside if you want to specify column sizes.

You can nest .flex, but a flex children can't be a flex itself.

Options

Sizes

Flex automatically add horizontal and vertical spacing to the contents.

Syntax Example
Class .flex-space-{size} .flex-space-small
Mixin .flex-space({size}) .flex-space(small)

You can set flex's sizes _variables.less file. Read the documentation of flex's variables.

.flex-space-none
.flex-space-tiny
.flex-space-small
.flex-space-medium
.flex-space-big
.flex-space-giant

Direction

Set children's wrapping inside the flexbox.

Syntax Example
Class .flex-{type} .flex-row
Class responsive .flex-{type}-{breakpoint} .flex-row-sm
Mixin .flex-{type}() .flex-row()
Mixin responsive .flex-{type}({breakpoint}) .flex-row('sm')
.flex-row
.flex-row-reverse
.flex-column
.flex-column-reverse
