--- title: Flex categories: [Layout] order: 200 description: Flex is used to place it's childrens in a column/row layout. You can set and all the flex's properties using utils classes/mixins. ---

Usage

Use .flex to create a column/row layout with flexbox.

Column

You have to use columns inside if you want to specify column sizes.

Syntax Example
Class .col
Class responsive .col-{breakpoint} .col-sm
Mixin .col()
Mixin responsive .col({breakpoint}) .col(sm)
You can nest .flex, but a flex children can't be a flex itself

Custom

To have custom width columns use .col-custom and width: Value; and max-width: Value.

Syntax Example
Class .col-custom
Class responsive .col-custom-{breakpoint} .col-custom-sm
Mixin .col-custom()
Mixin responsive .col-custom({breakpoint}) .col-custom(sm)

Variables

Read the documentation of layout's variables.

Options

Space

Flex automatically add horizontal and vertical spacing to the contents. You can specify a different one with this classes/mixins.

Syntax Example
Class .flex-space-{size} .flex-space-small
Class responsive .flex-space-{size}-{breakpoint} .flex-space-small-sm
Mixin .flex-space({size}) .flex-space(small)
Mixin responsive .flex-space({size}, {breakpoint}) .flex-space(small, sm)
You can set flex's spaces in the _variables.less file. Read the documentation of flex's variables.
