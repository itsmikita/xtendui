Read the documentation of Layout with theming and usage info.

Container

Containers are needed to center the website's contents and add spacing on the side. Normal containers .container have fixed width for each breakpoints, fluid containers .container-fluid have 100% of viewport width until max-width is reached.

You can set container's size and space in the _variables.less file. Read the documentation of grid's variables.

Grid

You can set grid's breakpoints in the _variables.less file. Read the documentation of grid's variables.

Columns are placed in a column/row layout using flex.
