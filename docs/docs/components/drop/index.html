Options

Variants

You can set default variant inside the .less file.

.primary .secondary .grey .white.

Sizes

.tiny .small .medium .big .giant.

Positions

You can set default position inside the .less file.

.bottom with .left .right .center.
.top with .left .right .center.
.middle with .left .right .center.
.before with .top .bottom .middle.
.after with .top .bottom .middle.
.middle-top and .middle-bottom.

Static

Width relative to parent's relative elements with .drop-outer.static.

Dimensions

max-height max-width.

Backdrop

.backdrop.
.backdrop-over.

Events

You can specify on and off events. Default is "on": "click".

Remember to use .backdrop-over on the toggle elements if you use backdrop.

"on": "mouseenter" "off": "mouseleave".

Nested

.bottom.left.
.bottom.left inside .drop-outer.static.
.after.middle-top.
.after.middle-top inside .drop-outer.static.

Animations

You can set default animation inside the .less file.
