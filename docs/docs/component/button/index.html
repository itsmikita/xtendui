Links

Links styles are defined inside /theme/_components.less.

On containers use .links-inverse to have white links.

In the code the mixins .link-underline() and .link-no-underline() are used assign proper underline on buttons.

Buttons

Buttons styles are defined inside /theme/_components.less.

After // styles you set shared styles, like font or border, after // options you set the default options for buttons, like size or special.

You set the button modifier classes after // size // special and // variant. You can remove or add new classes depending on your needs, and inside the classes you can override css as you wish (for example if you want all .btn-primary with a custom font).

Variants

You need to specify a variant on each .btn. -empty variants are for buttons without background.

.btn-link is styled like normal links but with custom color.

.btn-primary .btn-primary-empy .btn-secondary .btn-secondary-empy .btn-link

Sizes

Optional classes for assigning sizes. By default .btn-medium is used on all .btn. Sizes uses em padding that depends on font size.

.btn-tiny .btn-small .btn-medium .btn-big .btn-giant

Special

Special optional modifiers classes.

.btn-squared .btn-none .btn-icon .btn-tall .btn-short .btn-wide .btn-narrow

Custom
