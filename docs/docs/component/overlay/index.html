Usage

Use this markup to create a overlay. [data-xt-overlay] must be initialized by javascript.

For customizing the interactions read the documentation of toggle overlay.

Options

Size

Optional classes for assigning width.

Syntax Example
Class .overlay-size-{size} .overlay-size-small
Mixin .overlay-size({size}) .overlay-size(small)

You can set overlay's size in the _variables.less file. Read the documentation of overlay's variables.
